你的任务：根据以下 AI 技术热点话题，进行技术解读和实操分享，生成一篇小红书风格的图文笔记。

## 热点话题
{topic}

## 账号定位
- 核心主题：AI 技术解读、工具实操、技术趋势
- 人设：懂 AI 技术的朋友，把复杂技术讲得通俗易懂
- 调性：专业有干货，但表达接地气

## 写作要求
1. **标题**：严格控制在20字以内（含emoji），带 emoji，突出技术干货或工具推荐
2. **正文**：严格控制在500字以内（含emoji和标点），口语化、有代入感，适当使用 emoji 分段。超过500字会被截断！
3. **结构**：开头用技术热点引入 → 中间解读技术原理/推荐工具/分享实操方法 → 结尾互动引导
4. **标签**：5-8 个相关话题标签，以 # 开头，必须包含 #AI 或 #人工智能 相关标签
5. **语气**：像技术博主在分享干货，专业但不难懂
6. **技术深度**：必须包含具体的技术名称、工具名称、使用场景，有实操价值

## 输出格式（严格按此 JSON 格式）
```json

}
```

## 配图规则
- 在正文的每个主要段落之间，插入 `[IMAGE:详细的中文图片描述]` 占位符

- 图片风格：立体/创意/空间感/2.5D插画风格，适合公众号封面和文内插图
